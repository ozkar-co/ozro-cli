OzRo — parche cliente
=====================

1. Cierra el juego.
2. Extrae TODO sobre tu carpeta del cliente kRO (donde estan data.grf y renewal2021.grf).
3. Reemplaza archivos si Windows pregunta.
4. Abre OzRo.exe

Contenido:
  OzRo.exe, ozro.grf, gefenia.grf, data.ini
  System/itemInfo_EN.lua
  OzRo.ico (icono opcional para acceso directo)
