# Release binaries

Coloca aquí los archivos que van en `patch.zip` (no se generan desde el repo):

| Archivo | Origen |
|---------|--------|
| `ozro_patched.exe` | WARP / RSU sobre Ragexe.exe |
| `ozro.grf` | GRF Editor (fuentes en `data/`) |

`itemInfo_EN.lua` se genera automáticamente desde `data/System/itemInfo_EN.base.lua` + `itemInfo_custom.lua`.

Actualiza estos binarios cuando cambies GRF o exe; no hace falta tocar rutas externas del cliente.
